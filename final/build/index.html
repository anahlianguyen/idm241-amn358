    • Visual Focus Shift: Attention moves from the album cover to the spinning record, making the playback state clear.

Loops & Mode

Play Button
  • Loops
    • The hover animation can repeat infinitely each time the cursor enters or leaves.
    • Record Spin Loop: While the album is playing, the record on the back side continues spinning in an endless loop.
    • Flip State Loop: Each time the Play/Pause button is clicked, the album flips back and forth between the cover and the record, repeating the same smooth animation.
  • Mode
    • Play Mode: When the user clicks the Play button, the button switches to a Pause symbol, the album flips to the record side, and the record spins continuously.
    • Pause Mode: When clicked again, the button returns to the Play symbol, the album flips back to the front cover, and the record stops spinning.
    • Default Mode: Before any interaction, the Play button displays the Play symbol, and the album shows the front cover in a still state.

Download Icon

    • Pulse Stop: The pulse animation stops immediately.
    • Interaction Reset: The button can now be clicked again.
    • Database update: Removes downloaded songs to users Spotify profile.

Loops & Mode

Pause Button
  • Loops
    • Hover Loop: Each time the user hovers over the Pause button, it repeatedly glows and enlarges, then returns to normal when the cursor leaves—this happens every hover interaction.
    • Flip Loop: Every time the Pause button is clicked, the album consistently flips back to the front cover, repeating the same smooth animation.
    • Visual Reset Loop: The record consistently stops spinning and returns to a still state each time the Pause button is activated.
  • Mode
    • Active Mode (Pause State): When the album is playing, the Pause button is visible, the record is spinning, and the album is flipped to the back side.
    • Inactive Mode (Returns to Play State): When the Pause button is clicked, the button returns to the Play symbol, the album flips to the front GUTS cover, and the record stops spinning.
    • Default Mode: Before the user presses Play, the Pause button is not visible — the album appears still, showing the front cover with no animation.

Add to Library

    • The checkmark morphs back into a plus icon, returning to default state.
    • A reverse “fly out” animation sends the album cover back to its original position.
    • Database Update: Removes songs from user profile.
