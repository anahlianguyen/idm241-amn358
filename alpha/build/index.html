Rules

Artist Card
  • Hover On
    • The cursor changes from a pointer to a hand
    • The button background increases in opacity to indicate interactivity
    • The Play button animates upward, moving from the bottom of the card to the center
  • Hover Off
    • The cursor changes from a hand back to a pointer
    • The button background returns to its original opacity
    • The Play button smoothly moves from the center back down to the bottom and disappears
