Triggers

Album Preview Button
  • Hover On
  • Hover Off

Rules

Album Preview Button
  • Hover On
    • Cursor Change: The cursor changes from a pointer to a hand to indicate it's clickable.
    • Image Highlight: The rectangular album image gains a grey low opacity overlay.
    • Overlay Play Icon: A play icon becomes visible or increases in opacity at the center of the rectangle.
    • Tooltip: A tooltip appears above the element with the message: “Scroll through previews of tracks from this album.”
  • Hover Off
    • Cursor Change: The cursor changes back from a hand to a pointer.
    • Image Reset: The image returns to its original state.
    • Play Icon: The play icon returns to its default (hidden) state.
    • Tooltip: The tooltip disappears.

Triggers

Add to Library Button
  • Click (to activate)
  • Click again (to deactivate)

Rules

Add to Library Button
  • Click (to activate)
    • Button Press: The button animates as it's pressed down, providing tactile feedback.
    • Color Expansion: Green color expands from center to edges, indicating activation.
    • Confetti Burst: A brief confetti animation plays outward from the button's center.
    • State Change: Button remains green to show the active “downloaded” state.
  • Click again (to deactivate)
    • Button Press: The button reacts again to the click (press in and release).
    • Color Retraction: Green color collapses from edges back to the center, indicating deactivation.
    • State Reset: Button returns to its neutral base color and non-active state.

Feedback

Add to Library Button
  • Click (to activate)
    • Button Press: The button animates as it's pressed down, providing tactile feedback.
    • Color Expansion: Green color expands from center to edges, indicating activation.
    • Confetti Burst: A brief confetti animation plays outward from the button's center.
    • State Change: Button remains green to show the active “downloaded” state.
    • System / Backend: The album is added to the user's library in the backend.
  • Click again (to deactivate)
    • Button Press: The button reacts again to the click (press in and release).
    • Color Retraction: Green color collapses from edges back to the center, indicating deactivation.
    • State Reset: Button returns to its neutral base color and non-active state.
    • System / Backend: The album is removed from the user's library in the backend.
